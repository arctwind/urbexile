# Urbexile
**Official website:** [https://urbexile.com/](https://urbexile.com/)

## About

Urbexile is a text-driven roguelike urban survival game. It features a refined save system and an official mod framework that allows flexible expansion of events, enemies, and gameplay content.

## Download
Game builds are available in the [Releases](#) section.

## Modding
- The `ModSDK` folder includes documentation and sample mods.
- Only official API calls are supported; modders may not modify the core game.
- The Mod SDK and samples can be used to learn and create mods. **Commercial use is prohibited**, and open-sourcing your mods is **not required**.

## License
- **Core game**: Proprietary, all rights reserved.
  - Closed-source. Copying, modification, redistribution, or commercial use is strictly prohibited.
- **Mod SDK & Samples**: Free for non-commercial use only.
  - You may study, modify, distribute and create personal mods for non-commercial purposes.
  - Mods distributed via the official mod repository must be safe and reliable, and **must not contain viruses, malware, or disturbing/horror content**.
